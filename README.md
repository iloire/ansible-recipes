# ansible-recipes
My ansible recipes

# Linux/Ubuntu

- generate ssh key
- install git
- git clone repo
- install ansible using `install-ansible-linux.sh` or:
```
sudo apt update 
sudo apt install software-properties-common
sudo apt-add-repository --yes --update ppa:ansible/ansible
sudo apt install ansible
```
- run ansible

# Mac
- generate ssh key
- install git
- install homebrew manually (will automate it later some other time)
- install ansible (using pip or brew)
- git clone repo
- run ansible
